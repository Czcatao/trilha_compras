# Quicklist - Gerenciador de Lista de Compras

O Quicklist é uma aplicação web desenvolvida para facilitar a gestão de itens de compras semanais. A interface foca na usabilidade e na clareza das informações, garantindo uma experiência fluida tanto em dispositivos desktop quanto em dispositivos móveis.

## Funcionalidades

* **Adição de itens:** Permite incluir novos produtos na lista através de um campo de texto e um botão de confirmação.
* **Marcação de conclusão:** Opção de marcar itens como finalizados, aplicando um efeito visual de texto riscado.
* **Remoção de itens:** Exclusão de itens individuais da lista através de um botão específico.
* **Sistema de notificações:** Exibição de um alerta temporário no rodapé da página após a remoção de um item.
* **Design Responsivo:** Layout adaptável para diferentes resoluções de tela.

## Tecnologias Utilizadas

* **HTML5:** Para a estruturação semântica do conteúdo.
* **CSS3:** Para a estilização, utilizando conceitos de Flexbox e variáveis para padronização de cores.
* **JavaScript (ES6+):** Para a lógica de manipulação do DOM, controle de eventos e persistência da interface.

## Estrutura de Arquivos

* `index.html`: Contém a estrutura básica e os elementos da interface.
* `style.css`: Define as regras visuais, animações e responsividade.
* `script.js`: Gerencia a lógica de adicionar, remover e alternar o estado dos itens.

## Como Executar o Projeto

1. Salve os três arquivos (`index.html`, `style.css` e `script.js`) no mesmo diretório.
2. Execute o arquivo `index.html` em um navegador web de sua preferência.
3. Utilize o campo de entrada para começar a popular sua lista de compras.
